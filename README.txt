F74016255 謝孟成 資訊系105級
使用環境:
	ubuntu 64-bit Linux
執行:
	make 之後使用 ./give_question 和 ./solve
	分別去執行出題和解題
解題部分:
	全部初始化完後讀入題目，開始解題，遇到零時才開始解題，
	使用每直行橫列以及每九宮格去check如果還沒有某個數字表示有可能是這些數字，
	然後帶進去去跑，若錯誤再帶另外一個數字去跑，直到全部跑完。
	唯一解則輸出1以及解答，若無解則輸出0，多解則輸出2。
出題部分:
	使用以填好的a~i去做一個12*12的數讀map，然後再用random去讓a~i和1~9的random去match
	最後在挖空格並輸出。
